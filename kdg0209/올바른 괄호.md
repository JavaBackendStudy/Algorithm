## 문제 설명:

- 괄호가 바르게 짝지어졌다는 것은 '(' 문자로 열렸으면 반드시 짝지어서 ')' 문자로 닫혀야 한다는 뜻입니다.
- 
---

<br>

## 나의 문제 풀이:

```java
class Solution {
    boolean solution(String str) {
        	int index = 0;

		for (int i = 0; i < str.length(); i++) {
			char c = str.charAt(i);
			if (i == 0 && c == ')') {
				return false;
			}
			if (c == '(') {
				index++;
			}
			if (index != 0 && c == ')') {
				index--;
			}
		}

		return index == 0;
    }
}
```
