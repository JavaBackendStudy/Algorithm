---

## 문제 링크:

## 문제 설명:

- 문자 배열에서 특정 키워드의 인덱스를 찾은 후 결과값에 반환하여 출력한다.

---

## 나의 문제 풀이:

public class Solution {
    public String solution(String[] seoul) {
        return "김서방은 " + Arrays.asList(seoul).indexOf("Kim") +"에 있다";
    }
}
