# [프로그래머스] 특수문자 출력하기


---

## 문제 링크: https://school.programmers.co.kr/learn/courses/30/lessons/181948

<br/>

## 문제 설명:

- 다음과 같이 출력하도록 코드를 작성해 주세요.
- `!@#$%^&*(\'"<>?:;`
---

## 풀이 설명

1. 이스케이프 문자를 사용하여 특수문자 출력
2. \\:  역슬래시 문자 자체
3. \': 작은 따옴표(단일 인용부호)
4. \": 큰 따옴표(이중 인용부호)


```java
public class Solution {
    public static void main(String[] args) {
        System.out.println("!@#$%^&*(\\'\"<>?:;");
    }
}
```
